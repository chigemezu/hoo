JavaScript Cheat Sheet

Variables and Data Types

var: var x = 5; - Declares a variable with function scope.
let: let y = "hello"; - Declares a block-scoped variable.
const: const PI = 3.14; - Declares a constant variable.
Number: let num = 10; - Represents numeric data.
String: let str = "hello"; - Represents text data.
Boolean: let isTrue = true; - Represents true or false.
Array: let arr = [1, 2, 3]; - Represents a collection of elements.
Object: let obj = {name: "John", age: 30}; - Represents a collection of properties.

Operators

Arithmetic: +, -, *, /, %;
Assignment: =, +=, -=, *=, /=;
Comparison: ==, ===, !=, !==, >, <, >=, <=;
Logical: &&, ||, !;
Concatenation: + - For joining strings.

Functions

Function Declaration: function functionName() {};
Function Expression: let functionName = function() {};
Arrow Function: let functionName = () => {};
Arguments: functionName(arg1, arg2);
Return: return value;
Document: document.getElementById("myId"); - Returns an element by its ID.

Control Flow

If Statement: if (condition) {
  // code block
}
Else Statement: if (condition) {
  // code block
} else {
  // code block
}
Switch Statement: switch (expression) {
  case value1:
    // code block
    break;
  case value2:
    // code block
    break;
  default:
    // code block
}
For Loop: for (let i = 0; i < 5; i++) {
  // code block
}
While Loop: let i = 0;
while (i < 5) {
  // code block
  i++;
}
