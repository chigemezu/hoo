CSS Cheat Sheet

Selectors

* {} - Universal selector
element {} - Type selector
#id {} - ID selector
.class {} - Class selector
selector, selector {} - Grouping selector
selector1 selector2 {} - Descendant selector
selector > selector {} - Child selector
selector + selector {} - Adjacent sibling selector
selector ~ selector {} - General sibling selector

Functionalities

Box Model: How padding, border, and margin affect the size and layout of elements.
Positioning: Techniques for positioning elements on the page.
Flexbox: Layout model for creating flexible and responsive layouts.
Grid: Layout model for arranging elements in rows and columns.
Media Queries: Conditional styling based on device characteristics.
Transitions: Smooth animation effects between property changes.
Transforms: Apply 2D and 3D transformations to elements.
Animations: Create custom animations using keyframes.
Pseudo-classes: Style elements based on their state or position.
Pseudo-elements: Style specific parts of an element.
Gradients: Create gradient backgrounds.
Shadows: Apply box shadows to elements.
Responsive Design: Design websites that adapt to different screen sizes.
CSS Variables: Define reusable values within stylesheets.
Typography: Style text with different fonts, sizes, and spacing.
Grid Layout: Create complex layouts using CSS Grid.
