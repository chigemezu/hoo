HTML Cheat Sheet

Elements

<html> - Root element
<head> - Document Metadata
<title> - Title of the document
<meta> - Metadata
<link> - Relationship to external resource
<style> - Style information
<script> - Client-side script
<body> - Document content
<h1, h2, h3, h4, h5, h6> - Heading elements
<p> - Defines a paragraph
<a> - Defines a hyperlink
<img> - Embeds an image
<div> - Defines a division or section
<span> - Defines a section in a document
<ul, ol, li > - List elements
<table, tr, td, th > - Table elements
<input, form, button, textarea, label > - Form elements
<iframe> - Embeds another HTML page

Properties

src - Image source
alt - Alternate text for images
width - Width of the image
href - Hyperlink reference
id - Element identifier
class - Element class
style - Inline style
value - Value for form elements

Functionalities

Links - Create hyperlinks to other pages or resources
Images - Embed images with attributes
Forms - Create forms with various input types
Tables - Organize data in rows and columns
Lists - Create ordered and unordered lists
Styling - Apply styles using inline or external CSS
Text Formatting - Format text using HTML tags
Comments - Add comments in HTML code
Semantic HTML - Use semantic elements for better structure
Accessibility - Include attributes for accessibility
Responsive Design - Make the page responsive
Validation - Validate HTML code
HTML Entities - Use special characters
Audio/Video - Embed media content
IFrames - Embed external content
Local Storage - Store data on the client-side
